## 🐛 الملاحظات المفتوحة

> _لا توجد ملاحظات بعد. سيتم تسجيلها أثناء الاستخدام اليومي._

### [2026-07-13] لا يوجد نموذج إنشاء حساب في صفحة تسجيل الدخول

| الحقل           | القيمة     |
| --------------- | ---------- |
| **التاريخ**     | 2026-07-13 |
| **الصفحة**      | LoginPage  |
| **النوع**       | UX         |
| **مدى التأثير** | 🔴         |

**وصف المشكلة:**

صفحة تسجيل الدخول تحتوي فقط على نموذج تسجيل دخول (بريد + كلمة مرور). لا يوجد زر أو نموذج لإنشاء حساب جديد. المستخدم الجديد لا يستطيع الدخول للنظام.

**خطوات إعادة المشكلة:**

1. افتح `/login`
2. لا يوجد خيار "إنشاء حساب" أو "تسجيل"
3. المستخدم الجديد عالق

**ملاحظات إضافية:**

يحتاج النظام إلى tab للتبديل بين Login و Sign up.
